# GP_2: Сентимент новостей и ликвидность рынка

Часть 1: Новостной сентимент (NewsApi.ipynb)
Мы берём свежие новости через [NewsAPI](https://newsapi.org/), фильтруем по ключевым словам (финансы, рынки, макроэкономика) и прогоняем заголовки через **VADER Sentiment** из библиотеки `nltk`.  

**Что получаем на выходе:**
- Усреднённый compound score по всем новостям за день.
- Возможность отслеживать, как меняется тональность новостей со временем.
- Проверяем гипотезу о том, можем ли мы вообще пользоваться данным инстурументом.

В перспективе эти метрики можно использовать для построения регрессий или добавления как фичи в модели прогнозирования доходностей акций.

Часть 2: Анализ ликвидности CME LDB (LDB_analysis.ipynb)

Данные скачаны с портала [СЕЗАМ ВШЭ](https://sezam.hse.ru/) – это срезы биржевого стакана Chicago Mercantile Exchange за 2011 год.  
В папке `data/` лежат CSV объёмом ~5 млн строк.

1. **Загрузка и агрегация**  
   - Объединили все CSV в один DataFrame.  
   - Посчитали спред как `abs(ask1 - bid1)`.  
   - Сгруппировали по инструменту и дате, получили дневные агрегаты: средняя цена, цена закрытия, средний спред, объём.

2. **Очистка и разделение**  
   - Отделили ликвидные фьючерсы (`ES`, `CL`, `ZC`, `ZS`, `ZW`, `6E` и т.д.) от опционов и спредов.  
   - Убрали дни с аномальными скачками цен (|log_return| > 1.0).

3. **Внутридневной профиль ликвидности**  
   - Посмотрели, как меняется средний спред по часам торгов.  
   - Оказалось, что спред минимален в середине дня – хорошее время для исполнения крупных ордеров.

4. **Бэктест простой стратегии**  
   - Проверили «золотой крест» (MA5 / MA20) на ликвидных фьючерсах.  
   - Результат отрицательный: Шарп –10.6, убыток $18k. Вывод – трендовая стратегия в лоб на этих данных не работает.

5. **Факторный анализ (PCA)**  
   - Выделили главные компоненты дневных доходностей.  
   - PC1 объясняет ~45% дисперсии, сильно нагружена на зерновые (ZW, ZS, ZC) – это общий товарный фактор.  
   - Для диверсификации портфеля нужно добавлять активы из других классов (валюты, металлы).
